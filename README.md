# DigitalPathology_CNN-SVM

Задача решалась с помощью сверточной сети SimpleCNN + SVM по эмбеддингам (ансамбль CNN+SVM). CNN обучается с аугментациями, валидацией и early stopping, затем по 512-мерным эмбеддингам обучается SVM с RBF-ядром. При тесте используется усреднение вероятностей двух моделей.
